# 🔢 Multiple of 100 Checker

A simple Java program that reads an integer from the user and determines whether it is a multiple of 100.

This project was developed as part of a programming exercise focused on basic Java concepts such as data types, conditionals, and user input handling.

---

## ✨ Features

* 📥 Reads user input using `Scanner`
* 🔍 Checks if a number is divisible by 100
* ⚠️ Handles invalid input gracefully
* 🧩 Uses methods for better code organization

---

## 🧠 Concepts Applied

* Variables and data types
* Arithmetic operators (`%`)
* Conditional structures (`if-else`)
* Methods (modular programming)
* Input validation with `Scanner`

---

## 📄 How It Works

1. The program prompts the user to enter an integer.
2. It verifies whether the input is valid.
3. It checks if the number is a multiple of 100.
4. It displays the result in the console.

---

## ▶️ Example

**Input:**

```
12600
```

**Output:**

```
12600 is a multiple of 100
```

---

## 🚀 How to Run

1. Compile the program:

   ```bash
   javac MultiploDe100.java
   ```

2. Run the program:

   ```bash
   java MultiploDe100
   ```
